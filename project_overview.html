Project Overview


 Evrmore
A Peer-to-Peer System for Decentralized Finance Applications






Overview:
Evrmore is a blockchain project which is specifically targetted at Distributed Finance Applications. It is a code fork of Ravencoin, which itself enhanced the Bitcoin codebase and protocol to support assets. Evrmore is being built by current and former Ravencoin developers, hodlers, and miners as a parallel effort using a different set of trade-offs with different goals and different strategies. Evrmore seeks to improve upon Ethereum's failed security promises by further enhancing Ravencoin's asset protocol to enable Distributed Finance functionality WITHOUT general-purpose smart contracts and WITH the security of bitcoin.



Coin Specs:
          Algorithm: EvrProgPow (Programmable POW with 3GB starting DAG)

          Block Size: 2MB (may increase in the future)

          ICO: No

          Airdrop: Yes (one time, to all RVN on-chain holders at time of snapshot)
